
# Smart Study Recommender (AI/ML Project)

## Overview
This project uses Machine Learning to recommend which subject a student should focus on based on their scores in Physics, Chemistry, and Mathematics.

## Features
- Decision Tree ML model
- Model accuracy evaluation
- Data visualization
- Interactive web app using Streamlit
- GitHub-ready project structure

## Project Structure

smart-study-recommender
│
├── data
│   └── student_scores.csv
├── src
│   └── recommender.py
├── notebooks
│   └── analysis.py
├── model
├── app.py
├── requirements.txt
└── README.md

## Installation

pip install -r requirements.txt

## Train Model

cd src
python recommender.py

## Run Web App

streamlit run app.py
